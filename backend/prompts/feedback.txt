INPUT:
- Generated Content: Either a cover letter or HR question answer
- Job Description: The original job posting
- Filtered Profile Data: The job-specific profile information used

YOUR TASK:

- Tone adjustments (more/less formal, confident, enthusiastic)
- Better alignment with job requirements
- Clarity and conciseness improvements
- Missing emphasis on key skills or experiences
- Structural improvements

OUTPUT FORMAT (JSON only):
Return a JSON object with:
- feedback_items: Array of feedback objects, each with:
  - type: One of "tone", "alignment", "clarity", "emphasis", "structure"
  - suggestion: Specific, actionable improvement suggestion

1. Provide 2-4 specific, actionable feedback items
2. Use appropriate feedback types: tone, alignment, clarity, emphasis, structure
3. Focus on improvements that would make the content more effective
4. Be constructive and specific - avoid vague suggestions
5. Prioritize feedback that enhances job-fit and professionalism
